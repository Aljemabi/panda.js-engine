Download

  • Download Panda.js engine
    • If you don't have Node.js, download it now and install.

Install

  1. Open terminal
  2. Install Panda.js node toolkit
    3. $ npm install pandajs -g
  3. Unzip Panda.js
  4. Start web server
    5. $ cd panda.js-master
    $ panda server
    Starting Panda.js server...
    Server started at http://localhost:5975
  5. Open your browser at http://localhost:5975
  6. Start coding!
