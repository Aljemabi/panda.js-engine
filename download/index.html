Install from GitHub

  1. Download Panda.js engine
  2. Install local web server
  3. Unzip and open dev.html in your browser
  4. Start coding!

Or use NPM

$ npm install -g pandatool
$ mkdir MyPandaGame
$ cd MyPandaGame
$ panda install

Contribute

Feel free to contribute to the project at GitHub. If you find a bug, or have feature request, please use issue tracker.
