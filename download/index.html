Install from NPM

  1. Install Node.js
  2. Install git
  3. Install Panda.js toolkit
    4. $ npm install pandatool -g
  4. Install Panda.js engine
    5. $ panda install
  5. Start Panda.js server (optional)
    6. $ panda start
  6. Start coding!

Install from GitHub

  1. Download Panda.js engine
  2. Install local web server
  3. Unzip and open index.html in your browser
  4. Start coding!
