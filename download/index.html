Download

  1. Download Panda.js engine
  2. Download Node.js

Install

  1. Install Node.js
  2. Open terminal / command prompt
  3. Install Panda.js node toolkit
    4. $ npm install pandajs -g
  4. Unzip Panda.js
  5. Start web server
    6. $ cd panda.js-master
    $ panda server
  6. Open your browser at http://localhost:5975
  7. Start coding!
