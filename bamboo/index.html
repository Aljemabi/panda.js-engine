1. User interface

1.1 Overview
This is the main view of Bamboo editor. It contains toolbar, statusbar, property window, layer window and scene view.
1.2 Toolbar
Toolbar contains buttons with some main functions:
  • Add node
    • Add new node to scene.
  • Assets
    • View list of current assets in scene.
  • Save JSON
    • Save scene as JSON in media/ folder.
  • Save module
    • Save scene as module in game/scenes/ folder.
  • Download JSON
    • Download scene as JSON file.
  • Download module
    • Download scene as module JavaScript file.
1.3 Property window (node not selected)
When no node is selected, property window contains settings for scene:
  • Name
    • Name of scene.
  • Width
    • Width of scene in pixels.
  • Height
    • Height of scene in pixels.
  • Background color
    • Background color of scene.
1.4 Property window (node selected)
When node is selected, property window can be used to change node properties. You can view all node properties from API documentation.
1.5 Layers window
Layers window contains all your layers, as well as settings for selected layer:
  • Active node
    • Dropdown list of all nodes from selected layer.
  • Visible
    • Is layer visible on editor.
  • Fixed position
    • Does layer use fixed position (not moving with camera).
  • Name
    • Name of layer.
  • Speed
    • Speed of layer, can be used to create parallax effect.
There is also four buttons that you can use to add new layer, move layer up, move layer down and delete layer.
1.6 Statusbar
Statusbar shows your current editor mode and state, with all keyboard shortcuts.

2. Getting started

2.1 Installing Bamboo
  1. Download Bamboo ZIP from GitHub.
  2. Open editor.html in your browser.
2.2 Creating new scene
2.3 Changing scene settings
2.4 Adding new node
You can add new node to scene with Add node button or by pressing A from your keyboard. You will then see new Add Node window, where you can select your node type, parent and give it a name. Default parent is current selected layer. After pressing Add, position your new node by moving mouse, and confirm with click.
