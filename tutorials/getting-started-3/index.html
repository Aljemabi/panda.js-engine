Playing with sprites

Sprites are images that you can move, scale, rotate etc.

To make new sprite, you should first load image file for preloader:

game.addAsset('media/player.png');

You can create new sprite like this:

var player = new game.Sprite(100, 200, 'media/player.png');
game.Sprite needs three parameters: x, y, and path to image file.

Then you need to add sprite to stage or other container:

var player = new game.Sprite(100, 200, 'media/player.png');
game.scene.stage.addChild(player);

Here is a example of scene that adds one sprite on it's stage:

SceneTitle = game.Scene.extend({
    init: function() {
        var sprite = new game.Sprite(0, 0, 'media/player.png');
        this.stage.addChild(sprite);
    }
});

See all methods and properties of sprite from documentation.
