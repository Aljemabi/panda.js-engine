Getting to know Panda.js

Modules

Panda.js is made out of modules. Modules are basically JavaScript files. Modules are defined with module function.

Here is an example of simple module named game.main:

game/main.js
game.module(
    'game.main'
)
.body(function() {
    
});

Name of the module is also the path to the JavaScript file. For example, game/main.js is the file for the module above.

Every module must have body, and modules can also require other modules.

game/main.js
game.module(
    'game.main'
)
.require(
    'game.objects'
)
.body(function() {
    
});
You can require multiple modules. Separate module names with comma.

Classes

Game objects are made of classes. You can create new class by extending the base class:

MyClass = game.Class.extend({
    init: function() {

    }
});

If the class have init function, it is called, when you create new instance of the class, like this:

Player = game.Class.extend({
    init: function() {
        console.log('Hello player!');
    }
});

var player = new Player(); // Hello player!

You can also pass parameters to init function:

Player = game.Class.extend({
    init: function(name) {
        console.log('Your name is ' + name);
    }
});

var player = new Player('Panda'); // Your name is Panda

Classes can contain functions and properties:

Player = game.Class.extend({
    name: null,

    init: function(name) {
        this.name = name;
    },

    jump: function() {
        console.log(this.name + ' jumps!');
    }
});

var player = new Player('Panda');
player.jump(); // Panda jumps!

Your own classes can also be extended:

Player = game.Class.extend({
    name: 'Player',

    jump: function() {
        console.log(this.name + ' jumps!');
    }
});

Panda = Player.extend({
    name: 'Panda'
});

var player = new Player();
player.jump(); // Player jumps!

var panda = new Panda();
panda.jump(); // Panda jumps!

Extending functions is done with this.super function:

Player = game.Class.extend({
    name: 'Player',

    jump: function() {
        console.log(this.name + ' jumps!');
    }
});

Panda = Player.extend({
    jump: function() {
        this.name = 'Panda';
        this.super();
    }
});

var player = new Player();
player.jump(); // Player jumps!

var panda = new Panda();
panda.jump(); // Panda jumps!

Scenes

Scenes are special classes, that you need for your game to run. Your game need to have at least one scene.

You can create new scene by extending the game.Scene class:

SceneTitle = game.Scene.extend({
    backgroundColor: 0x808080,

    init: function() {

    }
});
Panda will by default start your game from scene SceneTitle

You can switch scenes with game.system.setScene command:

SceneTitle = game.Scene.extend({
    backgroundColor: 0x808080,

    init: function() {
        game.system.setScene(SceneGame);
    }
});

SceneGame = game.Scene.extend({ 
});

Every scene has update function, that is called every frame, when the scene is active.

Example

Here is a very basic example of a game, that contains one module with scene and player class:

game/main.js
game.module(  
    'game.main'  
)
.require(
    'engine.core'
)
.body(function() {  
     
Player = game.Class.extend({  
    name: null,  
  
    init: function(name) {  
        this.name = name;  
    }      
});
  
SceneTitle = game.Scene.extend({
    player: null,  
  
    init: function() {  
        this.player = new Player('Panda');  
    }  
});  
  
game.start();  
  
});
Note that you must require engine.core module for your game to work.
