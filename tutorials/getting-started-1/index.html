Installing Panda.js

1. Install web server

Before installing Panda.js, you need to have web server installed on your operating system. For more info about installing a web server, click here.

2. Download Panda.js

Download Panda.js from here. Unzip it to your web server folder.

3. Open browser

Navigate to the unzipped folder with your browser.

Example. If your web server is http://localhost and your unzipped folder is panda.js, the address will be http://localhost/panda.js/.

You should now see the Panda.js logo and you are ready to start developing HTML5 games!
