Making things interactive

Interacting with stage

Every scene has four predefined functions: click, mousedown, mouseup and mousemove.

These can be used for both mouse and touch events.

Every function get's object as a parameter, that you can use to get your mouse/touch position:

click: function(event) {
    var x = event.global.x;
    var y = event.global.y;
}

Here is example of scene with all four functions:

SceneTitle = game.Scene.extend({
    mousedown: function(event) {
        var x = event.global.x;
        var y = event.global.y;
    },
    mouseup: function(event) {
        var x = event.global.x;
        var y = event.global.y;
    },
    click: function(event) {
        var x = event.global.x;
        var y = event.global.y;
    },
    mousemove: function(event) {
        var x = event.global.x;
        var y = event.global.y;
    }
});

Making sprite interactive

Sprites are not interactive by default. To make them interactive, you need to define the functions that you need:

sprite.interactive = true;
sprite.click = function(event) {
}

You can also define everything in sprite's constructor function:

var sprite = new game.Sprite(100, 100, 'media/player.png', {
    interactive: true,
    click: function(event) {
        this.remove();
    }
});
