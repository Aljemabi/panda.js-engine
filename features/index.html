Main features

Canvas / WebGL

Pixi.js powered rendering with super fast speed.

Particle engine

Create stunning special effects using particles.

Tweening

Tween anything with easing, looping and grouping.

Physics engine

Hit detection and response with different shapes.

Timers

Add timers with callback functions and repeating.

Mobile support

Make games for mobile and tablet devices.

Sound manager

Play sound effects and music in your game.

Modules

Keep your code organized using modules.

Other features

Dynamic loader

Load assets and audio between scenes.

Sprite sheets

Package your sprites into one image file.

Animations

Supports sprite sheets, Spine and Adobe Flash.

Interactivity

Manage events for mouse and touch devices.

Bitmap fonts

Make great looking texts using bitmap fonts.

Plugins

Make and share your own modules as plugins.

Retina / HiRes

Support Retina and HiRes with auto image loading.

Packaging

Package your game into one minified JavaScript file.

Object pooling

Pool objects for better performance.

Local storage

Save and get data using browser local storage.

Debug draw

See your sprite hit areas and physic shapes.

Page visibility

Pause engine, when user switches to another tab.

Fullscreen

Play your game on a browser with fullscreen mode.

CocoonJS

Build your game for iOS and Android devices.

Windows Phone 8

Build your games for Windows Phone 8 devices.

Analytics

Track Google Analytics events inside your game.

Keyboard

Use your keyboard to interact with your game.

Sitelock

Prevent easy copying with sitelocking.

Accelerometer

Use device accelerometer to control your game.

Offline

Play your games offline without internet connection.
