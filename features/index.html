--- layout: default title: Features ---

Canvas/WebGL renderer

Panda.js uses Pixi.js for super fast rendering.

Preloader

Load all your assets and sounds before starting game.

Sprites

Sprite with support for alpha, rotation, scaling etc.

Sprite sheets

Use sprite sheets made with Texture Packer.

Spine/Flash animations

Export animations directly from Spine or Adobe Flash.

Tween engine

Tween anything from sprites to animations with easing and looping.

Mouse/touch events

Separate events for mouse and touch with multi-touch support.

Web Audio

Panda.js support Web Audio and HTML5 audio. Play, stop, loop, mute your sounds and music.

Particle engine

Use particles to make special effects.

Bitmap fonts

Make good looking text using bitmap fonts. Supports bmGlypgh.

Physics engine

Use our own physics engine for fast and easy hit detection and response.

Modules

Panda.js is built using modules, you can easily create and modify them.

Plugins

Make and share your own plugins.

Retina/HiRes

Panda.js supports Retina and HiRes modes with separate image files.

Object pooling

Use object pooling for better performance.

Local storage

Easily save highscore and other data to browsers local storage.

Debug draw

Draw all your sprites hit areas and physics shapes for easy debugging.

Page Visibility

Pause your game automatically, when user switches to another page tab.

Full Screen

Play your game on a fullscreen mode.

CocoonJS

Wrap your game as a native application for iOS and Android.

Windows Phone 8

Build your games for Windows Phone 8.

OUYA

Play your games on OUYA game console using CocoonJS.
