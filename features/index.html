Main features

Canvas/WebGL renderer

Pixi.js powered rendering with super fast speed.

Physics engine

Hit detection and response with different shapes.

Particle engine

Create special effects with particle engine.

Tweening

Tween objects with easing, looping and grouping.

Timers

Add timers with callback functions and repeating.

Mobile support

Make games for mobile and tablet devices.

Sound manager

Manage sound effects and music with automatic looping.

Modules

Split your game into modules and keep your code organized.

Other features

Preloader

Load all your assets and sounds with customizable preloader.

Sprites

Use images with position, alpha, rotation, scale and anchor.

Sprite sheets

Package your sprites into one image file for faster loading.

Animations

Use frame-by-frame animation or export from Spine or Flash.

Interactivity

Manage events for mouse and touches with multi-touch support.

Bitmap fonts

Make great looking texts using bitmap fonts.

Plugins

Make and share your own modules as plugins.

Retina/HiRes

Support Retina and HiRes modes with separate image files.

Object pooling

Use object pooling for better performance, specially on mobile.

Local storage

Easily save highscore and other data to browsers local storage.

Debug draw

See your sprite hit areas and physic shapes for easy debugging.

Page visibility

Pause the engine automatically, when game is not visible.

Fullscreen

Play your game on a browser with fullscreen mode.

CocoonJS

Wrap your game as a native application for iOS and Android.

Windows Phone 8

Build your games for Windows Phone 8 mobile.

Analytics

Use Google Analytics to track events inside your game.

Keyboard

Use your keyboard to interact with your game.

Sitelock
1.1

Lock your game into specific hostname to prevent easy copying.

Accelerometer
1.1

Use your devices accelerometer to control things in your game.
