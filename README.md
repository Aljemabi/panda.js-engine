## Panda.js HTML5 game engine

### Features

- Canvas/WebGL renderer ([Pixi.js](http://www.pixijs.com))
- Preloader
- Sprites/sprite sheets
- Flash animations
- Spine animations
- Tweening
- Mouse/touch events
- Web Audio API
- Particles
- Bitmap fonts
- Module system
- Retina/HiRes support
- Object pooling
- Local storage
- Page Visibility API
- CocoonJS compatible
- Windows Phone 8 compatible
- OUYA compatible
