Panda.js HTML5 game engine
==========================

![logo](https://raw.github.com/ekelokorpi/panda.js/master/media/logo.png)

## Features

- Pixi.js renderer
- Mobile support
- Retina / HiRes support
- Web Audio API
- Web Storage
- Object pooling
- Tween engine
- Physics engine
- Page Visibility API
- JSHint valid
- CocoonJS compatible
- Windows Phone 8 compatible
