Panda.js HTML5 game engine
========

## Features

- Pixi.js renderer
- Mobile support
- Web Audio support
- Web Storage
- Retina / HiRes
- Object pooling
- Tween engine
- Physics engine
- Page Visibility
- Module system
- Device and browser detection
- JSHint valid
- Mobile compatible
- CocoonJS compatible
- Windows Phone 8 compatible
