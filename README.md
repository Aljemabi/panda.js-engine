Panda.js HTML5 game engine
==========================

### Features

- Pixi.js renderer
- Mobile support
- Retina / HiRes support
- Web Audio API
- Web Storage
- Object pooling
- Tween engine
- Physics engine
- Page Visibility API
- JSHint valid
- CocoonJS compatible
- Windows Phone 8 compatible
